# Sistema de Controle de Chamados

Sistema interno desenvolvido para organização de atendimentos técnicos e controle operacional de suporte.

## Objetivo

Simular um ambiente básico de suporte técnico interno, permitindo gerenciamento simples de chamados e organização operacional.

## Tecnologias Utilizadas

- Python
- Flask
- SQLite
- HTML
- CSS

## Funcionalidades

- Registro de chamados
- Atualização de status
- Organização de tickets
- Estruturação de atendimentos
- Controle operacional básico

## Estrutura do Projeto

```bash
support-ticket-system/
│
├── app.py
├── requirements.txt
├── README.md
├── templates/
├── static/
└── database/
```

## Como Executar

Clone o repositório:

```bash
git clone https://github.com/tullius-caesar/support-ticket-system.git
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o projeto:

```bash
python app.py
```

A aplicação será iniciada localmente em:

```bash
http://127.0.0.1:5000
```
